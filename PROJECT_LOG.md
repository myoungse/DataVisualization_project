# Visualization of Sudden Unintended Acceleration (SUA) Accident Data

---

## 🚀 Project Overview
본 프로젝트는 자동차 급발진 (Sudden Unintended Acceleration, SUA) 사고에 대한 데이터를 수집, 정제 및 시각화하여 일반인들에게 급발진 사고의 패턴, 특징, 그리고 관련 정보를 보다 직관적으로 이해할 수 있도록 돕는 것을 목표로 합니다. 신뢰성 있는 뉴스 기사 데이터를 기반으로 사고 현황을 분석하고, 다양한 시각화 기법을 통해 급발진 사고의 원인, 피해 유형, 관련 차량 정보 등을 다각도로 탐색할 수 있는 인터랙티브 대시보드를 제공하고자 합니다.
